# Quick Start Guide

## Installation

1. Open terminal in the project folder
2. Run: `pip install -r requirements.txt`

## Running the Game

```bash
python main.py
```

## Basic Gameplay

1. **Move** with **WASD** keys
2. **Click mouse** to cast spells at cursor
3. **Press 1-3** to switch between spells:
   - **1**: Fireball (high damage)
   - **2**: Frost Bolt (fast, medium damage)
   - **3**: Lightning Strike (very fast, highest damage)

4. **Defeat all enemies** on the map
5. **Reach the blue exit** (bottom right) to progress
6. **Complete all 3 maps** to win!

## Game Features

- ✅ 3 unique maps with increasing difficulty
- ✅ 3 enemy types with different stats and AI
- ✅ 3 spell types with unique mechanics
- ✅ Player health, mana, and skill cooldowns
- ✅ Experience system
- ✅ Pause menu (Space key)
- ✅ Full keyboard and mouse controls

## Tips

- Use Frost Bolt to attack quickly while maintaining mana
- Save Fireball and Lightning for tough enemies
- Don't get too close to enemies - use ranged attacks
- Keep moving to avoid enemy attacks
- Reach the exit immediately after defeating all enemies to progress

Enjoy the game! 🎮
